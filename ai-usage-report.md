# AI Usage Report

## Tools Used

- ChatGPT

## How AI Was Used

ChatGPT was used as a support tool during the development of this assignment.

It helped with:
- suggesting interactive features suitable for the portfolio website
- improving the structure of the HTML, CSS, and JavaScript files
- explaining how to implement section switching using buttons
- helping with the dark mode feature and saving the theme using localStorage
- improving form validation and user feedback
- reviewing and improving the project documentation

## What I Changed Myself

I reviewed the suggested code and adapted it to fit my own portfolio website.

I changed the structure of the page, updated file locations, adjusted the content of the sections, and tested the final result to make sure the website worked correctly.

## Benefits of Using AI

Using AI helped me understand how to add more interactivity to the website in a simple way.

It also helped me improve the organization of the project and save time while learning how the features work.

## Learning Outcomes

Through this assignment, I learned:
- how to switch between sections using JavaScript
- how to save user preferences with localStorage
- how to validate a form and display feedback messages
- how to improve the user experience with transitions and hover effects
- how to review AI suggestions and apply them responsibly

## Responsible Use

AI was used as a learning and support tool only.

All generated suggestions were reviewed, tested, and adjusted before being included in the project.

I made sure to understand the purpose of the code and how each feature works before using it in the final version.
